Instructions for using Word Cloud Add On

How to download word cloud
Modern

Note that both allow saves in the .png format
Go to the modern version (see below)
You can now download the cloud by clicking on the download icon (see below). This will save it in your local downloads folder

Classic

In the classic version, when you run the cloud click on "make downloadable"
You will see the following screen
Then go on top of word cloud, right click and save as
Give a name to your word cloud. This will save it to a location of your choice on your computer
